Типы данных, которые поддерживаются 
компилятором напрямую, называются 
примитивными (primitive types) или 
встроенными у них существуют прямые 
аналоги в библиотеке классов .NET 
Framework Class Library

Framework .Net - единый каркас среды разработки
В каркасе Framework .Net можно выделить два основных компонента:

статический - FCL (Framework Class Library) - библиотеку классов каркаса;
динамический - CLR (Common Language Runtime) - общеязыковую исполнительную среду.

Метод Parse() в качестве параметра принимает строку и возвращает объект текущего типа
TryParse() пытается преобразовать строку к типу и, если преобразование прошло успешно, то возвращает true, иначе возвращается false

Объекты значимого типа существуют в двух формах: неупакованной 
(unboxed) и упакованной (boxed). Ссылочные типы бывают только в 
упакованной форме.
